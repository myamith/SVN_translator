Prerequisites & Steps:

1) Install SVN in local.
2) Install Python 2.7.
3) Install pyDEV in eclipse.
4) clone the project from GitHub.
5) change the source, dest paths to your local ones in config file.
6) complete the method read_parameters to read parameters from email (after txt file conversion).
7) change the source folders you want to copy.
8) copy file from dest location to PUME location.
